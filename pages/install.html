Установка Git и настройка GitHub

1. Установка Git

Git - это система контроля версий, которая позволяет отслеживать изменения в коде проекта.

  1. Скачайте установщик Git с официального сайта.
  2. Запустите установщик и следуйте инструкциям по установке. Выберите настройки по умолчанию или настройте их по своему усмотрению.
  3. После установки откройте терминал (командную строку) и проверьте версию Git с помощью команды:
    git --version

2. Настройка Git

Прежде чем начать использовать Git, укажите свое имя пользователя и адрес электронной почты:

  1. Откройте терминал и выполните следующие команды, заменив "Ваше Имя" и "your.email@example.com" на ваши реальные данные:
    git config --global user.name "Ваше Имя"
    git config --global user.email "your.email@example.com"

3. Работа с GitHub

GitHub - это платформа для хранения и совместной работы над проектами с использованием Git.

  1. Зарегистрируйтесь на сайте GitHub.
  2. Создайте новый репозиторий на GitHub, следуя инструкциям на сайте.
  3. Подключите локальный репозиторий к удаленному на GitHub с помощью следующих команд:
    git remote add origin https://github.com/ваш-пользователь/ваш-репозиторий.git
    git push -u origin master

Теперь вы готовы начать работу с Git и GitHub!
